# 今日头条——AI创作者辅助生产与分发平台
## 5.21更新日志
### 完成了界面风格确定
借鉴今日头条首页与登录窗口逻辑，整体布局借鉴抖音创作者中心风格，配色以今日头条红为主，辅以灰白色调，保持简洁现代感。以侧边栏为功能导航，红色突出按钮为本系统主要功能——即AI辅助内容创作页跳转按钮，然后是首页计划集成热点与爆文/推荐榜单模块，无限滚动应以**虚拟列表**实现。
### 完善用户登录流程
后端采用nodejs的express架构，使用腾讯云短信服务实现验证码发送功能，设计数据库用户表，通过jwt令牌实现用户登录状态持久化，前端localstorage存储token由zustand的presist中间件来实现。

前端Nextjs没有做代理，4000端口与后端4001端口跨域，在Nextjs中做代理，修改next.config.ts，实现api转发走同源：
```ts
async rewrites() {
    return [
        {
        source: "/api/:path*",
        destination: "http://localhost:4001/api/:path*",
        },
    ];
},
```
